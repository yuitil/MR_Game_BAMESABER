# MR_Game_BAMESABER

作品名：BEAM SABER(ビームセイバー)

制作期間：三ヶ月

制作人数：「N_」から始まるファイルは自分の担当で「A_」がもう一人のプログラマーです。サウンド担当と、エフェクト・モデリング担当の4名で作成しました。

使用エンジン：アンリアルエンジン

使用言語：C++

---

## 【ゲーム概要】
突如自分の部屋に地球侵略を目指す宇宙人が侵入してくるので、その宇宙人をビームセイバーを駆使して追い返すMRゲーム

---

##【こだわっった点】
プレイヤーに剣(セイバー)で弾を跳ね返す体験をしてほしいと考えたので、剣で弾を跳ね返す処理を一番こだわりました。
剣と弾が当たった時に、なんでもかんでも跳ね返すのではなく、プレイヤーが狙って跳ね返した感覚を持ってもらうために、
プレイヤーの目線に近い場所に敵がいる時にその敵に跳ね返るようにしました。
また作業感が出ないように、剣を振ったスピード取得したり、スイートスポット（野球のバットの芯）を作り跳ね返る弾に
強弱をの種類を作りました。
